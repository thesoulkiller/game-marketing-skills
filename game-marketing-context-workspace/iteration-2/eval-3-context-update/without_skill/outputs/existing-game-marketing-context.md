# Game Marketing Context

- **Genre**: Mid-core Tactical RPG
- **Platform**: Mobile (iOS / Android)
- **Stage**: Pre-launch / Soft launch

- **Age Range**: 18–35
- **Gender**: Broad male skew
- **Psychographic**: Strategy enthusiasts, RPG fans, competitive players
- **Geos**: United States, Germany, United Kingdom
